| Operator  | Function                      |
|-----------|-------------------------------|
| +         | addition                      |
| -         | subtraction                   |
| *         | multiplication                |
| /         | division                      |
| **        | exponential                   |
| abs()     | absolute value                |

```vhdp
Result := ValA * ValB;
Result := abs(Result);
```
