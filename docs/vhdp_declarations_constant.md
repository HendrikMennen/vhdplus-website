## Definition

Used as parameter in Main and Component or as parameter for the whole project in Package. Can be used like a Signal but is read only.

## Declaration

**Structure:**
```vhdp
CONSTANT <Name> : <Type> := <value>;
```
**Example:**
```vhdp
CONSTANT Width : NATURAL := 8;
```
